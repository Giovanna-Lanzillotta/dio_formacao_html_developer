Estrutura Básica do HTML

A estrutura básica do HTML é composta pelas Tags: html,head e body. Ela organiza o conteúdo e diz ao navegador a ordem e como será exibido.Sua estrutura é lida de cima para baixo,ou seja ele começa a lê pela tag !DOCTYPE html.

Tags

As Tags são as instruções que o navegador deve seguir para formatar um texto.A maioria delas começa com uma de abertura e outra de fechamento.O conteúdo deve ficar entre elas para poder funcionar. Exitem vária tags e algumas possuem atributos.

Tags: h1 a h6

Funcionam como um cabeçalho.Começando peço h1 e terminando no h6 o tamanho vai dimunuido conforme o numero aumenta.

Tag: mark

Serve para marcar ou realçar algumas partes ou toda parte de um texto. Ele lembra um marca-texto fluorescente.Sua cor padrão é amarela.

Tag: small

Usada para diminuir uma parte do texto em relação ao tamanho do texto princial.

Tag: i

Transforma a parte do texto que está entre as tags em itálico.

Tag: u

Ela serve para sublinhar a parte do texto que esta entre as tags.

Tag: strong

É usada para realçar uma parte muito importante no texto. Ele se parece com o negrito, mas tem significado totalmente diferente.

Tag: font

Define caracteristicas para o texto como cor, tamanho e fonte.Não é muito usado, pois é preferível usar o CSS.

Tag: del

Indica se houver uma revisão ou modificação em um texto qeee que foi removido ou edititado intencionalmente. É colocado uma linha rizxcadi riscada na frete da palavra que será corrigida.

Tag: p

Indica um parágrafo de um texto.

Organiza um texto com quebra de linhas.Cada texto que estiver entre as tag p forman um parágrafo diferente.

Tag: abbr

É umatag de abreviação.

HTML Linguagem de Marcação de Hipertexto.

RJ Estado do Brasil

Ao passar o mouse por cima da abreviação você pode ver o significado da sigla.

Tag: sub

É usada para fazer um texto subscrito. Pode ser bem utilizada em matemática e/ou formulas químicas e até em notas de rotapés e referências.Exemplos:

A fórmula da água H2O

log636 = 2

Tag: sup

É usada para fazer um texto sobrescrito. Pode ser bem utilizada em matemática, em notas de rotapés e referências.Exemplos:

32 = 9

A área do quarto é de 42m2.

Tag: blockquote

Indica um bloco de texto, como por exemplo uma citação.

Segundo Alan Turing:

Um computador merecia ser chamado de inteligente se pudesse enganar um humano fazendo-o o acreditar que era humano.
Isto é um exemplo de como a tag blockquote aparece em um texto.

Atributos das Tags

Os atributos das Tags são palavras especiais dentro das tags que acrescentam caracteristicas a elas modificando dua forma de exibição.Geralmente elas ficam na tag de abertura, ou seja no começo

Existem vários tipos de Tags como:

  • Atributos Globais

    • id:Define um identificador.
    • class:Define uma classe.
    • style:Permite adicionar estilos do CSS direto dtag no html.
    • title:Define um texto quando o mouse é passado sobre o texto.

Textos

As tags de textos servem para modificar elementos de texto no HTML.

Listas Ordenadas e Listas Não Ordenadas

Listas Ordenadas

Apresentam Listas em ordem ordenada.São representadas pela tag ol que iniciam e encerram as Listas e li que representam cada item.

  1. Item 1
  2. Item 2
  3. Item 3

Listas Não Ordenadas

Apresentam Listas que não possui uma ordem específica.São representadas pela tag ul que iniciam e encerram as Listas e li que representam cada item.

  • Item 1
  • Item 2
  • Item 3
